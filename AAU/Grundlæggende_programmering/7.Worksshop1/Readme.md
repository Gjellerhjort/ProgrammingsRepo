# run program
## local envirment
```bash
pip install -r requirements.txt

python main.py
```
## virtual envirment

### windows
```bash
python -m venv venv

.\venv\Scripts\activate

pip install -r requirements.txt

python main.py
```

### linux 
```bash
python -m venv venv

source ./venv/Scripts/activate.sh

pip install -r requirements.txt

python main.py
```
